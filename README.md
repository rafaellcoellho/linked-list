# Linked List

A simple linked list implementation for a [blog post](https://rafaellcoellho.netlify.com/2019/02/22/lista-ligada-linked-list-em-c/).

## Prerequisites

Just install [ceedling](https://github.com/ThrowTheSwitch/Ceedling).

## Usage

Compile and run

```bash
$ meson build
$ cd build
$ ninja
$ ./src/main
```
